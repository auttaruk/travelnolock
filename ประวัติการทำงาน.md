## 2026-05-27 (ครั้งที่ 3) — แก้ 3D Scanmap + เพิ่มเมืองใหม่

**2. แก้ `setScanCountry()` — ก่อนหน้าไม่โหลด OSM เลย**
- อัปเดต `sandboxLat/Lng` ให้ตรงกับเมืองที่เลือก
- ตั้ง `walkStartLat/Lng` ใหม่
- เรียก `fetchRealNearbyAttractions(preset.lat, preset.lng)` อัตโนมัติ
- เรียก `map.setView()` ให้แผนที่ fly ไปเมืองนั้น

**3. แก้ `toggleScan` — ก่อนหน้าไม่โหลด OSM ตอน Start**
- เรียก `fetchRealNearbyAttractions()` ทันทีเมื่อกด Start Scan

**4. Overhaul `fetchRealNearbyAttractions()` — query แคบเกินไป**
- เปลี่ยนจาก `[tourism]` เดี่ยวๆ เป็น union query รวม:
  - `[tourism]` + `[historic]` + `[amenity~"restaurant|cafe|bar|..."]` + `[leisure~"park|garden|..."]`
- เพิ่ม `_typeLabels` dictionary แปลประเภทเป็นภาษาไทย
- กรองเฉพาะ node ที่มีชื่อจริง (ไม่ขึ้น `Spot #N`)
- Fallback เรียบร้อย: ถ้า OSM ว่างเปล่าหรือ timeout → ใช้ hardcoded `sightseeingSpots`
- คำอธิบายและ review ดีขึ้น ไม่ใช่ placeholder

## 2026-05-27 (ครั้งที่ 2) — เพิ่ม Entry Guide Skill + ปรับปรุงระบบ

### งานที่ทำ
